Project Resources

You've learned a lot. So it's time to put it to good use! Here's a list of resources I've found while working on my own projects.

Where to Find APIs

RapidApi is a hub for finding all sorts of APIs. Be careful and check the docs of any you are interested in.

Google API Explorer is the place to find any API made by Google. Google Maps? Google Books? Right here.

The Reddit API is a surprisingly robust and user-friendly API for tons of data from Reddit. I know, the docs look like garbage. But they are super helpful.

If you think of a website that might have a data set you need (like Fandango or Spotify), try googling the website name with "api." You'd be surprised what's out there.

CSS Frameworks

Bulma works a lot like Bootstrap. The classes have similar naming conventions and the columns system for responsive layout design will be pretty familiar. Bulma is pretty simple and doesn't have any fancy components that require JavaScript. It's so simple to use, I used it to make this!

MaterializeCSS was made by Google and works a little bit differently from Bootstrap or Bulma. It doesn't have a system for handling responsive layouts. But it does have a lot of super cool components that use JavaScript with CSS for some really sleek UI. If you've got a handle on media queries and want to challenge yourself, check it out!

Semantic UI is an extremely robust framework with some of the best docs I have ever used. It has more components than MaterializeCSS or Bulma. Like MaterializeCSS, it uses JavaScript to handle some of the flashier stuff. Semantic UI doesn't have a column system the way Bulma or Bootstrap does.

Design Help

Coolors is an easy way to make cohesive color palettes. Having, and sticking to, a specific color scheme will step up your websites and keep your branding consistent.

This gradient generator allows you to select two colors and grab the CSS you need to put it on your page. It also has pre-made buttons and a few other elements to explore.

Pexels is a place to get license-free images for your websites. You can browse by category, color, vibe, whatever you might need. Protip: start with an image you really like for your site and build the color palette around it.

Google Fonts. Because if I see Comic Sans on your website, I will scream.
